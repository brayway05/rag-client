

## Client: React + TypeScript + Vite + [TailwindCSS](https://tailwindcss.com/)

*Vite is a speedy Javacript bundler for building the React app

Commands to develop locally:

`npm install`

`npm run dev`

`docker run -p 8080:80 -d rag-client`
